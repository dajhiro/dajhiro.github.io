## Mapping from CSS
### Box & Layout
`display: flex` <> `flex`

`flex-direction: row` <> `flex-row`
`flex-direction: column` <> `flex-col`

`justify-content: center` <> `justify-center`
`justify-content: space-between` <> `justify-between`

`align-items: center` <> `items-center`

`width: 100%` <> `w-full`
`height: 100%` <> `h-full`
`min-height: 100vh` <> `min-h-screen`

### Padding / Margin
`padding: 16px` <> `p-4`
`padding: 24px` <> `p-6`

`padding-left/right: 16px` <> `px-4`
`padding-top/bottom: 16px` <> `py-4`

`margin: 16px` <> `m-4`
`margin-top: 32px` <> `mt-8`
`margin-buttom: 8px` <> `mb-2`

### Text
`font-size: 14px` <> `text-sm`
`font-size: 16px` <> `text-base`
`font-size: 20px` <> `text-lg`

`font-weight: 500` <> `font-medium`
`font-weight: 600` <> `font-semibold`
`font-weight: 700` <> `font-bold`

`text-align: center` <> `text-center`
`color: #111` <> `text-gray-900`
`color: #666` <> `text-gray-500`

### border / background
`border: 1px solid #eee` <> `border border-gray-200`
`border-buttom: 1px solid #ddd` <> `border-b border-gray-300`
`border-radius: 8px` <> `rounded-lg`
`background: whit` <> `bg-white`
`background: #f9fafb` <> `bg-gray-50`

### 기타
`position: fixed` <> `fixed`
`top: 0` <> `top-0`
`left: 0` <> `left-0`
`z-index: 10` <> `z-10`
`cursor: pointer` <> `cursor-pointer`
`overflow: hidden` <> `overflow-hidden`
