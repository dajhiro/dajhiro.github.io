## 작업 스케줄러: Quartz 자동화
### 작업 스케줄러: 파일 생성
`AutoSync.bat`
```powershell
# AutoSync.bat
@echo off
cd "C:\Users\kuzed\quartz"
npx quartz sync
# pause
```

### 작업 스케줄러: 추가
순서: **작업 스케줄러** → **작업 만들기**
- **일반**
	- **이름**: Quartz Sync Automation
	- **사용자 로그온 여부와 관계없이 실행**: 백그라운드 설정
- **트리거**: 새로 만들기
	- **작업 반복 간격**: 10분
- **동작**: 새로 만들기
	- **프로그램/스크립트**: `C:\Users\kuzed\AutoSync.bat`

### 작업 스케줄러: 현재 스케줄 확인하기
![[../../게임 프로젝트/attachment/Pasted image 20250808134034.png|500]]
- 왼쪽 사이드바에 **작업 스케줄러 라이브러리**를 클릭하면 현재 스케줄들이 나옴
- 해당 스케줄러를 선택 후에 우측 사이드바 **실행**을 클릭하면 테스트 진행 가능
