## Conventional Commits
```Makefile
feat: 로그인 화면에 자동완성 기능 추가
fix: 회원가입 시 이메일 중복 체크 버그 수정
docs: README에 설치 방법 추가
refactor: 불필요한 변수 제거
style: 코드 포맷팅 수정 (기능 변경 없음)
chore: 의존성 패키지 업데이트
test: 유닛 테스트 추가
```
