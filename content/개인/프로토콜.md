### L5
세션: 두 호스트 간의 대화 단위, 
- 이 요청과 저 응답이 같은 맥락이다.

로그인 상태 유지, 파일 전송 재개, [[RPC]] 호출

TCP가 못하는 것
- 멀티플렉싱: 하나의 TCP 연결 위에서 여러 독립적인 대화를 동시에 구분하는 것
- 체크포인팅: 전송이 끊겼을 때 처음부터가 아니라 중간부터 재개
- 양방향 동기화: 양쪽이 동시에 말할 수 있는 full-duplex 대화 조율

⇒
- 세션 식별: HTTP Cookie / Session ID
- 멀티플렉싱: HTTP/2 스트림, [[QUIC]] 스트림
