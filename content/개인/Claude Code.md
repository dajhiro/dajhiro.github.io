### `/batch` — 대규모 병렬 마이그레이션

사용자와 인터뷰를 진행한 뒤, 수십-수백 개의 worktree 에이전트로 작업을 분산한다. 각 에이전트는 독립 Git worktree에서 전체 테스트를 수행하고 독립 PR을 생성한다. 대규모 코드 마이그레이션에 적합하다.
