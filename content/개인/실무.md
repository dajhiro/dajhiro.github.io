CRC
Cyclic Redundancy Check
순환 중복 검사

CRC값: 원본 데이터를 특정 다항식으로 나눈 나머지
송신 측 값과 수신 측 값과 대조

CRC 에러: 두 값이 동일하지 않음
=> 데이터 변형

복구는 재전송 또는 별도의 오류 정정 코드(ECC)

CRC 에러 발생 영역
- 네트워크: 이더넷 프레임 손상, 케이블 불량
	- CRC errors, input errors
- 디스크/스토리지: 배드 섹터, 디스크 노후화
	- Data error (cyclic redundancy check)
- 압축 파일: 다운로드 중 손상, 불완전 전송
	- CRC failed, CRC mismatch
- 직렬 통신: UART, RS-485 등 노이즈 간섭
	- 프레임 단위 CRC fail
- 메모리/펌웨어: 플래시 영역 손상
	- firmware CRC error
