리눅스 퍼미션 모델
```
[type][owner][group][other]
        rw-    r--   r--
```

type bit
- `-`: 일반 파일
- `d`: 디렉토리
- `l`: 심볼릭 링크
- `s`: 소켓
- `p`: 파이프
- `c`: 문자 디바이스
- `b`: 블록 디바이스

sudo 정책
```bash
# /etc/sudoers.d/dev-team
# 개발자: 앱 재시작만
%dev    ALL=(root) NOPASSWD: /bin/systemctl restart myapp
%dev    ALL=(root) NOPASSWD: /bin/journalctl -u myapp

# 배포팀: 배포 스크립트만
%deploy ALL=(root) NOPASSWD: /opt/scripts/deploy.sh

# DBA: PostgreSQL 관련만
%dba    ALL=(postgres) NOPASSWD: /usr/bin/psql
```
