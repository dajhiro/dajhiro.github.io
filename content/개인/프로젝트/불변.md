# Immutable: Mutable에 대응
There are two types of objects:
- immutable
- mutable


- immutable: it has a unique object id, so it can't be changed.
	- its **state** can be changed if it is a mutable object.
- mutable: 

	
- 주소, 돈
- 값을 바꾸려면 새로운 객체를 만들어서 교체

Immutable
```java
class Money {
    private final int amount;  // final로 한번 정해지면 못 바꿈
    
    Money(int amount) { this.amount = amount; }
    
    Money add(int value) {     // 새 객체를 만들어서 반환
        return new Money(this.amount + value);
    }
    
    int getAmount() { return amount; }
}
```

```java
Money basePrice = new Money(1000);
Product productA = new Product("기본 티셔츠", basePrice);
Product productB = new Product("프리미엄 티셔츠", basePrice);

Money premiumPrice = basePrice.add(500); // 새 객체 생성, basePrice는 그대로
productB.setPrice(premiumPrice);

System.out.println(productA.getPrice().getAmount()); // 1000
System.out.println(productB.getPrice().getAmount()); // 1500
```
