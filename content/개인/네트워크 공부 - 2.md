OSI 7 Layer, OSI 7계층

OSI, Open Systems Interconnection model
각 계층은 서로 독립적, 특정 기능 수행.

1번 물리 계층, Physical Layer
- 케이블, RJ45 커넥터, 허브

2번 데이터 링크 계층, Data Link Layer
- 프레임 단위로 데이터 구성, MAC 주소를 사용하여 데이터 전송

3번 네트워크 계층, Network Layer
- 다른 네트워크 간의 데이터 전송 담당: IP 주소 등

4번 전송 계층, Transport Layer
- 두 장치 간의 신뢰할 수 있는 데이터 전송 보장: TCP/UDP 프로토콜 

5번 세션 계층, Session Layer
- 네트워크 상에서 통신하는 두 장치 간의 세션을 관리
- 통신의 시작과 종료

6번 표현 계층, Presentation Layer
- 데이터 형식 정의, 암호화 및 압축 담당
- 파일 인코딩, 데이터 압축 등

7번 응용 계층, Application Layer
- 사용자와 직접적으로 상호작용하는 계층
- 웹 브라우저, 이메일 클라이언트 등의 응용 프로그램

---
### 키워드
- 프레임(Frame)
	- 데이터 링크 계층에서 데이터를 전송하는 단위
	- 헤더와 트레일러로 구성되어 데이터의 시작과 끝을 나타내며, 오류 검사 정보를 포함
- Mac 주소(Media Access Control Address)
	- 네트워크 장비가 세계적으로 유일한 식별자를 갖도록 하는 물리적 주소
	- 데이터 링크 계층에서 사용
- IP 주소(Internet Protocol Address)
	- 인터넷 상에서 컴퓨터나 장비를 식별하는 주소
	- 네트워크 계층
- 전송 계층
- [[TCP]](Transmission Control Protocol)
	- 신뢰성 있는 데이터 전송
- UDP(User Datagram Protocol)
	 - 빠른 전송
