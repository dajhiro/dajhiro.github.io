## Segmentation Fault
프로그램이 허용되지 않은 메모리 **영역**에 접근 시도, 
또는 허용되지 않은 **방법**으로 메모리 영역에 접근 시도.

Segmentation
운영 체제에서 사용하는 메모리 관리 및 보호의 한 기법

메모리 관리 정책: 페이징으로 대체

일부 운영 체제: 논리 레벨에서 세그멘테이션 사용
- 유닉스: `SIGSEGV` 잘못된 메모리에 접근하는 프로세스에 신호 전달
- 윈도우: `STATUS_ACCESS_VIOLATION`

## Bus Error
하드웨어에 의해 발생하는 [[폴트]]fault
중앙 처리 장치가 물리적으로 **주소 할당을 할 수 없는** 메모리에,
프로세스가 접근을 시도한다는 것을 운영 체제에 알린다.

[[주소 버스]]의 유효하지 않은 주소
POSIX 준수 플랫폼: `SIGBUS` 신호 발생, 해당 프로세스에 전달
