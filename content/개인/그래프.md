- 사용자가 여러 그룹에 속하거나

	- 트리는 "부모는 하나만"이라는 제약을 걸었을 뿐
- 태그

조직/HR 구조
트리: 전통적 보고 체계(팀장-팀원)
그래프:
- 매트릭스 조직: 한 사람이 기능 조직과 프로젝트 조직 양쪽에 소속
- 점선 보고: dotted-line reporting
- 협업 네트워크 분석

파일 시스템
- symbolic link, hard link, mount point(UNIX)

	- 한 파일이 여러 경로에서 참조될 수 있다.

	- 어떤 컴포넌트가 어떤 모듈을 import 하는지
		- 순환참조도 가능

네트워크
- 그래프: 물리적 네트워크 토폴로지
- 스패닝 트리 프로토콜
- 


단위: node
계층 관계와 네트워크 관계의 차이

트리
- 최상위 노드
- 하나의 부모
- 단방향(순환 x)
- 유일경로
- 예: 파일 시스템, 조직도, 가계도, 분류 체계

그래프
- 동등한 노드
- 여러 부모 연결
- 유연한 방향성(순환 o)
- 여러 경로
- 예: 소셜 네트워크, 지도(도로), 지식 그래프, 웹 페이지 링크 구조

트리 $\subset$ 그래프
