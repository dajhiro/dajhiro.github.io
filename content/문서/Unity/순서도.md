```scss
(에디터/코드에서)
게임 오브젝트 생성 및 컴포넌트 추가
        │
        ▼
  ┌───────────────┐
  │    Reset()    │ ← 컴포넌트가 에디터에서 새로 추가될 때 1회 호출
  └───────────────┘
        │
        ▼
(씬이 로드되거나 인스턴스화될 때)
  ┌───────────────┐
  │    Awake()    │ ← 씬에 로드되자마자 1회 호출 (활성화 여부 무관)
  └───────────────┘
        │
        ▼
  ┌───────────────┐
  │  OnEnable()   │ ← 활성화 시마다 호출 (최초 활성화 포함)
  └───────────────┘
        │
        ▼
  ┌───────────────┐
  │    Start()    │ ← 활성화 상태 첫 프레임 전에 1회 호출
  └───────────────┘
        │
───────────── 게임 플레이 루프 시작 ──────────────
        │
        ▼
  ┌───────────────┐
  │  FixedUpdate()│ ← 물리 프레임마다 호출 (고정 주기)
  └───────────────┘
        │
        ▼
  ┌───────────────┐
  │   Update()    │ ← 매 프레임마다 호출 (게임 로직 처리)
  └───────────────┘
        │
        ▼
  ┌───────────────┐
  │  LateUpdate() │ ← Update()가 모두 끝난 뒤 호출 (카메라 등 후처리)
  └───────────────┘
        │
        ▼
  ┌───────────────┐
  │  OnDisable()  │ ← 컴포넌트 비활성화 시 호출 (이벤트 구독 해제 등)
  └───────────────┘
        │
        ▼
(게임 오브젝트 파괴 or 씬 언로드)
  ┌───────────────┐
  │  OnDestroy()  │ ← 오브젝트 또는 컴포넌트가 완전히 파괴될 때 호출
  └───────────────┘

```
